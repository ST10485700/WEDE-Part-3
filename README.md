 Project Overview

Kick Exquisite is a multi-page sneaker retail website designed to showcase products, sneaker culture, and provide users with cart, checkout, contact and enquiry functionality.

The site includes advanced JavaScript features, full SEO implementation, form validation, and responsive modern styling.

 Website Pages

The final website includes 8 fully completed pages:

product.html	Sneaker catalogue with add-to-cart system

cart.html	Cart summary with quantity control
checkout.html	Order summary + customer details form
information.html	Shipping, refund & store policies
contact.html	Contact form with validation
